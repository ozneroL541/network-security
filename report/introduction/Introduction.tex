\chapter{Introduction}
\label{chap:introduction}

The experience will be focused on the vulnerability assessment of a system, 
using the OpenVAS/Greenbone tool. 
The main goal is to identify and evaluate 
potential security weaknesses in a target system.. 
This process involves scanning the system for known 
vulnerabilities, analyzing the results, and providing recommendations for 
remediation. 
The experience will also cover the configuration and usage of 
OpenVAS/Greenbone, as well as best practices for conducting vulnerability 
assessments effectively.
In our scenario we will be assessing a network of vulboxes,
which are intentionally vulnerable containers, which simulates
a Local Area Network (LAN) with multiple hosts.
